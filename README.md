# Weather-Tracker
Fill in a city name and see the weather for that city!
You can look up any city and see the weather for the past 24 hours, the current weather and you can see the weather for the coming 24 hours.
Made possible with the Visual Crossing Weather API.
To test the app, simply open the HTML file with a web browser.
To view the code, open the files with a code editor.
Link to assignment: https://roadmap.sh/projects/weather-app
